# DreamKeeper-ARPG
3D-ARPG游戏框架 Unity-Framework

•	独立设计开发的原创游戏框架，持续开发中

•	游戏循环，资源加载，战斗系统，Json配置文件，Xml存档，UI框架，Log日志，消息传递

•	AI行为树，RPG背包，动画帧事件，场景状态机，多语言本地化……
